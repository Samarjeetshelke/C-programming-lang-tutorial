**Strings**

Strings are the collection of characters allocated in continue memory block and ended with **NULL**  or  **\0** .

**functions Used with string**:-
|       Function     |  Purpose                    |
|-----------------|-----------------------------|
|       strcpy(s1, s2);          |          Copies string s2 into string s1.      |
